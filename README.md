# Task 4: Classification with Logistic Regression

## Objective
Build a binary classifier using Logistic Regression to predict passenger survival on the Titanic.

## Dataset
Titanic Dataset (891 rows, 12 columns)

## Steps
1. Preprocessed data (handled missing values, encoded categorical features)
2. Split into train/test (80/20) and standardized features
3. Trained a Logistic Regression model
4. Evaluated using confusion matrix, precision, recall, ROC-AUC
5. Tuned classification threshold and explained the sigmoid function

## Results
- Accuracy: 81%
- ROC-AUC: 0.88
- Key features: Sex, Pclass

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
